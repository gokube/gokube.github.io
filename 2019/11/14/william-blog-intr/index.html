本blog安装、部署和使用方式

参考：install page

安装步骤：
--> 本地hugo命令安装: brew install hugo
--> 本地建立新site: hugo new site /path/to/site 
--> 本地新建md文档: hugo new post/william.md 
--> git安装themes：hugo-icarus-themes 
--> 修改 config.toml文件
--> 启动: hugo server
--> 同步: 在github建立仓库，然后hugo 生成public目录；最后push到github
